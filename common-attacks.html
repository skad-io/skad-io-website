Common Attacks

So many attack attempts!
Find out what the hackers are trying to do.

We've had our own Watch Dogs by SKAD up and running for a few months now. Over that time we've seen a lot of attacks. The vast, vast majority come in bursts from the same source, as a bot at that location comes upon our IP address and tries a bunch of attacks against it.

The majority of these attacks are attempts to log in with a known default username and password combo. Lots of hardware and software products come with default usernames and passwords already set up. Attacks on these are why it's always important to change any default passwords to something different and complex.

Why complex? Because a lot of people use passwords based on simple dictionary words. If a default username is known, a bot can run through a dictionary attack, using dictionary words to try to guess the password.

From the attacks we've already seen, here are explanations for a few.

The same attacker keeps trying with different usernames

They are either running through a list of common usernames or through a list of usernames they've gotten from a hack or leak.

The same attacker keeps trying the same username

They might be running a dictionary attack in an attempt to guess the password for that username from a list of dictionary words.

Most common usernames under attack

These usernames are common across many systems and devices, which is why they get tried a lot: admin, administrator, root, user, test, tester, support, guest.

root

The root user is the most powerful user on a Linux- or UNIX-based system. Quite a few IoT devices are based on these operating systems. If a hacker cracks root, they have full access.

PlcmSpIp

This is the default username for Polycom phones. Polycom phones also use the username Polycom.

ubnt

This is the default username for a Ubiquiti UniFi Access Point.

pi

Lots of Raspberry Pi distros use this default username – often with the password 'raspberry'. If you're opening a Raspberry Pi up to the internet, make sure you cahnge the password.

A list of UK towns

The same source keeps trying to log in using one UK town after another as the username.
We have no idea what this is about! But we'll update here when we do.

kodi

This is the default username for Kodi, a media server.

osmc

This is the default username for OSMC, another media server.

plexuser

This is the default username for plexuser, yet another media server.

uucp, nuucp, lpd

These are service usernames created by software installed on computers that have the AIX operating system. If you have an AIX computer, we'll eat our hats – so con't worry about this one.

ftp, ftpuser

These are default usernames for file transfer protocol servers. A lot of businesses use these to receive file from outside their business.

joggler

This is the default username for O2's Joggler tablet.

vyatta

This is the default username for Vyatta, a software-based virtual router.

...and many more

We'll add the interesting ones as we see them.
